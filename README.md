# 🔢 Base Converter (PyQt5)

A simple desktop application to convert numbers between different bases (2–16).

---

## 🚀 Features
- Validate input numbers
- Convert between different bases
- Clean PyQt5 interface
- Supports letters (A–F for hexadecimal)

---

## 🖥️ Technologies
- Python
- PyQt5

---

## ▶️ How to run

```bash
pip install -r requirements.txt
python main.py
